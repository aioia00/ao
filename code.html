閉じ込めた「わたし」を迎えにいく旅

私たちは長い間、「闇」を悪いもの、克服すべきものとして扱ってきました。しかし、その扉を開く鍵は、すでにあなたの手の中にあります。

シャドウ・アルケミー

抑圧されたエネルギーをほどくと、何が起こるのでしょうか？
心に閉じ込めた「影」をタップして、それが輝く「ギフト（才能）」へ姿を変える様子を体感しましょう。

心の声・シャドウチェック

知らず知らずのうちに、心の奥底（地下室）に押し込めてしまったエネルギーを診断してみましょう。 以下の「心のつぶやきやサイン」で、思い当たるものをチェックしてください。

感情を認める「解放のノート」

「解放」とは、衝動のままに行動することではありません。
心の中で「そう感じていたんだね」と、ありのままを認め、許すこと。 心の地下室に置いたままの想いを書き出し、光へと還しましょう。

闇とは敵ではない。闇とは、まだ迎えに行っていない自分自身なのである。
